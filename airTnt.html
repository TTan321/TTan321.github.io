AirTnt

AirTnt is an Airbnb inspired website where users are able to upload listings with one photo, and review book other user's listings.

AirTnt is a personal full stack web development project

This project was created to demonstrate my capabilities of working with Javascript (node) and other various frameworks by having a functional live website.

The tech stack used to create AirTnt is a node.js backend that utilized express.js to create Rest API endpoints and Sequelize to query a PostgreSQL database. React.js is used to build the frontend of AirTnt along with Redux to manage state.
